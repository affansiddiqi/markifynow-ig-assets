# MarkifyNow IG Assets

Public image host for Instagram API publishing. Instagram's Graph API requires
media to be fetched from a public URL — this repo exists only to serve that
purpose. No captions, strategy, or business data live here; those stay in the
private [linkedin-autopilot](https://github.com/affansiddiqi/linkedin-autopilot)
repo. This repo is mirrored automatically from there.

Images are served via raw.githubusercontent.com URLs, e.g.:
`https://raw.githubusercontent.com/affansiddiqi/markifynow-ig-assets/main/images/day01.png`
